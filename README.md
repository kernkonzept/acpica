# Lib-acpi for L4Re

This package contains the acpica libraries together with make files for
building them inside the L4Re tree.

The original acpica library is available at
[Github](https://github.com/acpica/acpica.git)

# Documentation

This package is part of the L4Re Operating System Framework. For
documentation and build instructions please refer to
[l4re.org](https://l4re.org).
